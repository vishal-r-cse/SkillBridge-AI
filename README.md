# SkillBridge AI

## Problem Statement

Students often struggle to identify the gap between their current skills and industry requirements.

## Solution Overview

SkillBridge AI helps students analyze resumes, detect skills, identify missing skills, and provide career guidance.

## Features

- Resume Upload
- PDF Resume Parsing
- Skill Detection
- Skill Gap Analysis
- Learning Roadmap
- AI Insights
- Career Readiness Score
- Download Report

## Technology Stack

- Python
- Streamlit
- PyMuPDF

## AI Tools Used

- ChatGPT
- Resume Skill Detection Logic

## Setup Instructions

pip install -r requirements.txt

python -m streamlit run app.py
